\Alorel\Dropbox\Operation\Files\UploadSession

\Alorel\Dropbox\Operation\Files\UploadSession Start

Upload sessions allow you to upload a single file using multiple requests. This call starts a new upload session with the given data. You can then use upload_session/append to add more data and upload_session/finish to save all the data to a file in Dropbox.



A single request should not upload more than 150 MB of file contents.
